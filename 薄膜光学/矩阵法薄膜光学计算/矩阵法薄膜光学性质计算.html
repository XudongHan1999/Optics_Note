                        修正导纳

                        • s 光：ζ~j=Njcos⁡θj
                        • p 光：ζ~j=Njcos⁡θj
                        xxxxxxxxxx
                        def eta_s_p(n ,theta ,mode):
                            if mode == 'p':
                                eta = n / cos(theta)
                            else:
                                eta = n * cos(theta)
                            return eta

                        薄膜特征矩阵

                        [cos⁡δjiζ~jsin⁡δjiζ~jsin⁡δjcos⁡δj]
                        xxxxxxxxxx
                        def M_matrix(n_d_data ,wavelength ,mode):
                            M_e = np.eye(2,2)
                            for row in n_d_data.itertuples():
                                theta = asin(n_0 * sin(theta_0)/row[1])
                                delta = 2 * pi/wavelength * row[2] * cos(theta)
                                eta = eta_s_p(row[1] ,theta ,mode)
                                M_e = M_e @ np.array([[cos(delta)       ,1j * sin(delta)/eta],
                                                      [1j * eta * sin(delta)    ,cos(delta)]])
                            return M_e

                        膜系特征向量

                        [BC]={∏j=1K[cos⁡δjiζ~jsin⁡δjiζ~jsin⁡δjcos⁡δj]}[1ζ~G]
                        xxxxxxxxxx
                        def v_vector(n_d_data ,wavelength ,mode):
                            M_e = M_matrix(n_d_data ,wavelength ,mode)
                            v_e = M_e @ np.array([[1] ,
                                                  [eta_s]])
                            return v_e

                        物理量计算

                        组合导纳

                        Y=CB

                        反射率

                        r=ζ~0−Yζ~0+YR=r~r~∗=(ζ~0B−Cζ~0B−C)(ζ~0B−Cζ~0B−C)∗

                        透射率

                        T=4Re⁡[ζ~0ζ~G](ζ~0B+C)(ζ~0B+C)∗

                        吸收率

                        A=1−R−T
                        xxxxxxxxxx
                        def calculate(v_e):
                            B = v_e[0] ;C = v_e[1] 
                            Y = C/B
                            r = (eta_0 - Y)/(eta_0 + Y)
                            R = (abs(r)**2).real
                            T = (4 * (eta_0 * eta_s).real/(abs(eta_0 * B + C)**2)).real
                            A = (1 - R -T).real
                            return (R ,T ,A)

                        看图说话

                        为方便讨论，引入 λ/4 膜，即

                        ndcos⁡θ=mλ4

                        m 为奇数（λ/4膜层），为反射率极值点

                        R=(n0−Yn0+Y)2=(n0−n12nGn0+n12nG)2

                        m 为偶数（λ/2膜层），为“虚设层”（反射率与膜层折射率无关），

                        R=(n0−nGn0+nG)2

                        n>nG，为增反膜，并且有效光学厚度为 λ/4 奇数倍时，反射率达到极大值

                        n<nG，为增透膜，并且有效光学厚度为 λ/4 奇数倍时，反射率达到极小值

                        n=n0或n=nG，膜层不存在

                        对于 λ/4 膜层，当膜层折射率满足 n=n0nG，反射率 R 为零

                        高反带与高透带

                        {g<1波段⇒不再有高反带出现g>1波段⇒高透、高反带周期性间隔出现

                        高反带与高透带中心

                        • 高反带中心 g=1,3,5⋯
                        • 高透带中心 g=2,4,6⋯

                        高反区半宽度

                        Δg=2πarcsin⁡(nH−nLnH+nL)
                        • 所有高反带的波数宽度相等，波数宽度仅与两种膜料折射率差值有关
                        • Δλ=λ2λ0Δg，各高反带波长宽度不相等，波数越大，波长宽度越窄

                        注意：垂直入射时，（s光，p光）波数宽度无差别，斜入射时则不相等

                        高反射膜与周期多层膜系

                        利用周期多层膜系制成的高反射膜可使反射率 R>99%，且具有较宽的带宽。要达到这样高的反射率，单层膜是无法实现的，通常采用光学厚度均为 λ/4 的高、低折射率交替排列构成的周期性多层膜，以下是常见的高反射膜系结构

                        • A | (HL)m | G →Y=(nHnL)2mnG
                        • A | (HL)mH | G →Y=(nHnL)2mnH2nG
                        • A | (LH)m | G →Y=(nLnH)2mnG
                        • A | (LH)mL | G →Y=(nLnH)2mnL2nG
                        R=r~r~∗=(n0−Yn0+Y)2nH>nL，m→∞，R→1

                        周期多层膜系理论

                        周期多层膜系由两种均匀介质膜层周期性排列构成，其周期性条件表示为

                        n(z)=n(z+h)

                        其中 h 为一个周期两膜层的总厚度，假设单周期膜层特征矩阵为

                        Q(h)=[q11q12q21q22]

                        则 m 个周期构成的膜系特征矩阵为

                        Q(mh)=[Q11Q12Q21Q22]=[Q(h)]m

                        如果特征矩阵 Q(h) 的行列式取单位值，根据矩阵论 （研究生阶段的一门课程，不关线性代数的事）

                        Q(mh)=[q11Um−1(a)−Um−2(a)q12Um−1(a)q21Um−1(a)q22Um−1(a)−Um−2(a)]Um(x)=sin(m+1)arccos⁡x1−x2，|x|⩽1（第二类切比雪夫多项式）a=12(q11+q22)

                        也就是说膜系特征矩阵的元素 Q11、Q12、Q21、Q22 与切比雪夫多项式的取值相关，切比雪夫多项式的取值由 a 决定，a 的变化（nL 与 nH 的变化，相对波数 g 的变化）特点将会反映出膜系反射率的变换特点。对于 [HL]m 类型的周期多层膜其变换特点已在上面图像中显示。
